# Learning Dashboard 🚀

A modern student learning dashboard built using Next.js, Supabase, Tailwind CSS, and Framer Motion.

---

## 📌 Features
- Server-side data fetching using Supabase
- Animated Bento grid layout
- Course progress tracking
- Smooth Framer Motion animations
- Responsive design (mobile + desktop)

---

## 🛠️ Tech Stack
- Next.js (App Router)
- Supabase
- Tailwind CSS
- Framer Motion
- TypeScript

---

## 🚀 Getting Started

```bash
npm install
npm run dev
